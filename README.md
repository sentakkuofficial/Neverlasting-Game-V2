# NEVERLASTING: ORIGINS

A bullet-heaven survival game set in the *Owari No Nai — Neverlasting* world.
You are Ichido. The death tournament is over — now the black wind of KuroKaze
comes for everything you have left. Auto-attack, collect embers, level up, and
outlive the swarm as long as you can.

Single file. No dependencies. No build step.

## Play
- **Move:** WASD or arrow keys (touch joystick on mobile)
- **Weapons fire automatically.** Collect glowing **embers** to level up.
- **Each level:** pick 1 of 3 upgrades — new abilities or stronger ones.
- **P** to pause. Bosses (Vidar, the Headmaster) arrive after the first minute.

### Abilities (all drawn from the lore)
Kaichi Bolt · Bronze Chain · Daichi Quake · Tokasu's Vial · White Scarf · Ember Storm
plus passives: Vigor, Swiftness, Focus, Haste, Greed, Resolve, Fortune.

## Deploy to GitHub Pages (1 min)
1. Create a new repo and upload `index.html` (and this README).
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Branch: `main`, folder: `/ (root)`. Save.
4. Your game is live at (https://sentakkuofficial.github.io/Neverlasting-Game-V2/).

That's it — `index.html` at the root is served automatically.
